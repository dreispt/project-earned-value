=========================
Project Earned Value Task
=========================

This module allows you to manage the progress of your projects using the
Earned Value Management technique. For a detailed explanation, see the
`Slides <https://www.odoo.com/es_ES/slides/slide/project-management-with-the
-lights-on-earned-value-management-using-odoo-283>`_ and `Video <https://www
.youtube.com/watch?v=lvCodX1TK5E>`_ of the conference at Odoo Experience
2015.
